## 安装方式

安装完成重启HA，刷新一下页面，在集成里搜索`云音乐`即可

[![Add Integration](https://my.home-assistant.io/badges/config_flow_start.svg)](https://my.home-assistant.io/redirect/config_flow_start?domain=ha_cloud_music)
